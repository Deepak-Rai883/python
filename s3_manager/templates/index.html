AWS s3 Manager

prepared by: Deepak Rai

Discription:

This python project is about building a web app that can manage our AWS S3 buckets with basic functionalities that are as following:
1. listing the available buckets
2. creating and deleting buckets
3. Listing the files inside the buckets
4. uploade/ download/ delete files inside folders
5. copy or move files between two buckets
